# Mental Health Prediction using Machine Learning

## 📌 Project Description
This project predicts whether a person requires mental health treatment using machine learning techniques.

## 📊 Dataset
Mental Health Survey Dataset (Kaggle)

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## 🤖 Algorithm
Logistic Regression

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Upload the dataset (mental_health.csv)
3. Run all cells
4. Get prediction results

## 📈 Output
The model predicts whether a person needs mental health treatment based on input features.
